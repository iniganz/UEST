# Images Folder

Folder ini untuk menyimpan gambar-gambar website:

## File yang diperlukan:

1. **logo.png** - Logo tournament (recommended: 100x100px atau lebih)
2. **favicon.png** - Icon untuk browser tab (recommended: 32x32px atau 64x64px)
3. **hero-bg.jpg** - Background untuk hero section (recommended: 1920x1080px)

## Tips:

- Gunakan format PNG untuk logo (support transparansi)
- Gunakan format JPG untuk background (ukuran file lebih kecil)
- Compress gambar sebelum upload menggunakan [TinyPNG](https://tinypng.com) atau [Squoosh](https://squoosh.app)

## Sumber Gambar Gratis:

- [Unsplash](https://unsplash.com) - Foto HD gratis
- [Pexels](https://pexels.com) - Foto & video gratis
- [Canva](https://canva.com) - Buat logo sendiri
- [Freepik](https://freepik.com) - Vector & ilustrasi

## Catatan:

Jika file gambar tidak ada, website tetap berjalan normal. Logo akan diganti dengan text.
